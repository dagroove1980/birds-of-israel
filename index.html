To use this app, you need to add your eBird API key:

  1. Get your API key from eBird API Key Generator
  2. Create a file named config.js in this directory
  3. Add: window.EBIRD_API_KEY = 'your-api-key-here';
  4. Refresh this page

For GitHub Pages: Since config.js is not included (for security), you'll need to fork this repository and add your own API key.
